# Day 2
##Learned git commands
- git init - initializes local repo for git to track
- git status - see changes or stages
- git add - stage all changes, to add to next commit
- git commit -m get the snapshot of the changes locally
- git push -u origin main  sends changes/commits to github
- git branch tells u what branch ur on
- git log shows all commits author date message
- git remote add origin [url] and git remote -v links github repo to git and tells if its connected
-git add [. = all] or [file = specific file]
